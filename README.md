# AI-Powered Travel Planning System – Lanka Travels

## Overview

AI-Powered Travel Planning System – Lanka Travels is a full-stack intelligent travel planning platform designed to generate personalized travel recommendations and itineraries based on user preferences, budget, and trip duration. The system integrates machine learning techniques, web technologies, and database management to provide a seamless travel planning experience.

This project was developed as part of an undergraduate Artificial Intelligence and Machine Learning coursework project.

---

## Key Features

### Personalized Travel Recommendations

* Generate customized travel itineraries based on user preferences.
* Recommend destinations according to budget and travel duration.
* Provide intelligent travel suggestions using machine learning techniques.

### Destination Management

* Browse and explore tourist destinations.
* View destination details and travel information.
* Organize destinations into personalized trip plans.

### Trip Planning & Scheduling

* Create day-by-day travel itineraries.
* Manage travel schedules and activities.
* Optimize travel plans based on user requirements.

### Ticket Management

* Create and manage service tickets.
* Track ticket status and updates.
* Perform complete CRUD operations for ticket handling.

### User Management

* User registration and authentication.
* Secure login and profile management.
* Session management and access control.

### AI & Machine Learning Integration

* Data preprocessing and cleaning.
* Dataset preparation and feature engineering.
* Model training and evaluation.
* Recommendation generation using machine learning algorithms.

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AI-Powered-Travel-Planning-System.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Flask Application

```bash
python app.py
```

### Access Application

```text
http://localhost:5000
```
